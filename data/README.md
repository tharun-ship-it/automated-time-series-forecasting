# Dataset

## PJM Hourly Energy Consumption

This project uses the PJM Interconnection hourly energy consumption dataset.

### Download Instructions

1. Visit [Kaggle - Hourly Energy Consumption](https://www.kaggle.com/datasets/robikscube/hourly-energy-consumption)
2. Download `PJME_hourly.csv`
3. Place it in this `data/` directory

### Dataset Description

- **Source:** PJM Interconnection LLC
- **Region:** PJM East
- **Frequency:** Hourly
- **Period:** 2002-2018
- **Records:** 145,366

### Features

| Column | Description |
|--------|-------------|
| Datetime | Hourly timestamp |
| PJME_MW | Energy consumption in megawatts |

### Alternative

If you can't download from Kaggle, the notebook includes fallback sample data for demonstration purposes.
